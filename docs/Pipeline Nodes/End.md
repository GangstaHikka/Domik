=== "Description"
	An **End** node is a final node of the pipeline. 
	You should have one to work with the Pipeline properly.  
=== "Inputs and Outputs"
	- **Cells** - cells to be built  
	- **Override Skin Layers** - cells which should have special skin instead of own.
