# **Filter Floor**

![[Pasted image 20210918154450.png]]{ align=left }

=== "Description"
	Use this node to filter concrete floor.  
	
=== "Inputs and Outputs"
	- **In** - cells to filter
	- **Out** - filtered cells
	- **Floor** - input number (optional).
	- **Number field** - Number of the floor to filter
